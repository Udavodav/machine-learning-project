# machine-learning-project

## Проект: Предсказание оттока клиентов телеком-компании

## Цель проекта
Разработка модели машинного обучения для предсказания оттока клиентов телеком-компании. Модель позволяет выявлять клиентов с высокой вероятностью ухода для своевременного применения удерживающих мер.

## Основные задачи
- Анализ и подготовка данных о клиентах телеком-компании
- Построение моделей бинарной классификации с учетом дисбаланса классов
- Оптимизация гиперпараметров моделей для достижения максимальной точности
- Калибровка вероятностей для получения достоверных оценок
- Создание ансамблей моделей для улучшения предсказательной способности
- Выбор оптимальной модели для развертывания


## Быстрый старт

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd <project-directory>
```

### 2. Установка зависимостей
```
pip install -r requirements.txt
```

### 3. Подготовка данных
```
cd notebooks
python clear_data.py
```

Этот скрипт выполнит:

- Загрузку исходных данных
- Очистку от пропущенных значений
- Кодирование категориальных признаков
- Масштабирование числовых признаков
- Балансировку классов (3 варианта: unbalanced, balanced_medium, balanced)
- Сохранение обработанных данных в `data/processed/`

### 4. Обучение моделей
```
python models.py
```


Этот скрипт выполнит:

- Загрузку подготовленных данных
- Оптимизацию гиперпараметров для Random Forest и Gradient Boosting
- Обучение моделей на трех наборах данных
- Калибровку вероятностей
- Создание ансамблей (Voting и Stacking)
- Оценку моделей на тестовых данных
- Сохранение лучших моделей в `models/`

### 5.  тестов
```
pytest tests/ -v
```
